Typography

타이포그라피

Font선택은 가독성에 우선하며, 지정된 서체와 크기를 부여함으로서 사이트 전반의 일관성 유지할 수 있도록 한다.

기본준수사항

비주얼 영역에 들어가는 캠페인 문구등은 나눔고딕 서체를 사용한다.
기본 권장 폰트 이외의 그래픽 폰트 사용시 사이트의 성격에 맞는 폰트 3가지 이하를 선정하여 일관되게 사용하며, 사용전 시스템 소유의 계열사에 폰트 저작권 여부를 확인한다.

타이포그라피

  1. 사이트 전반에 사용되는 본문 색상은 Web color: #4d4d4d를 기본으로 하되, 배경색상에 따라 달라질 수 있다.
  2. ‘돋움, Verdana, Arial’을 기본 타입으로 사용하며, 나눔고딕이나 나눔명조와 같은 라이센스 프리 폰트를 사용할 수 있다.
  3. 본문 폰트를 CSS에서 표현할떄 아래와 같은 기준을 따른다.
    • - 폰트사이즈의 기본은 12px(0.86em)이며 최소 8PX(0.57em)보다 작게 사용하지 않는다.
    • - 문단의 행간은 가독력을 높이기 위해서 “line-height: 18px 또는 최소 160%이상”값을 주도록 한다.(게시판 컨텐츠를 포함한 본문 컨텐츠 모두 해당)
    • - 하이퍼 링크의 색상은 “검정색(black : #000000), 사용자가 한번 이상 방문했던 링크의 색상은 보라색(purple : #686868)으로 하며, 커서가 올라갔을 때의 링크 색상은
    • - 사용자 템플릿에 적합한 칼라로 차별화되어 적용한다.
    • - 나눔 폰트 사용할 경우 사용자가 PC최초 접속시 폰트를 설치할 수 있도록 폰트 다운로드 소스를 추가한다.

다국어 기본 서체

  • 한국어돋움, dotum, Helvetica, sans-serif
  • 영어Arial, Helvetica, sans-serif
  • 중국어宋体, “Arial Narrow”
  • 일본어MS PGothic, “Osaka”, Arial, sans-serif
  • 베트남어Arial
  • 인도네시아어Arial, Helvetica, sans-serif
  • 포르투갈어Helvetica Neue, Helvetica, Arial, sans-serif
