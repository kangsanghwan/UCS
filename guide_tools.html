평가도구

웹표준 - KW3C 2.0(한국인터넷진흥원)

웹사이트가 제공하는 웹 콘텐츠의 DTD(문서 형식 정의) 준수 여부를 평가해주는 도구

  • - 표준 CSS 문법 준수진단 (W3C CSS Validator)
  • - 표준 (x)HTML 문법 준수진단 (W3C Markup Validator)
kw3c2.0.2 다운로드 kw3c2.0.2 사용방법

주요 기능

  • - 웹표준 평가 기능 (W3C 웹표준 문법진단, W3C 스타일 시트 문법진단)
  • - 평가 대상 (온라인 검사: URL 원격시스템 검사, 오프라인 검사: 파일 로컬시스템)
  • - 프로젝트 관리 (프로젝트 생성, 재편집, 백업 등)
  • - 평가결과 보기 (평가URL/소스/오류 목록 연동기능)

웹표준 - Korea HTML5 사이트(한국인터넷진흥원)

홈페이지 URL 및 파일을 통하여 홈페이지에 존재하는 액티브X, 웹 표준, 웹 연결성, 웹 개방성에 대한 진단 결과를 제공.

  • - 표준 CSS 문법 준수진단 (W3C CSS Validator)
  • - 표준 (x)HTML 문법 준수진단 (W3C Markup Validator)
Korea HTML5 사이트 이동

주요 기능

  • - 액티브X 진단
  • - 웹 표준 진단
  • - 웹 연결성 진단 (HTML 소스내부 링크 및 앵커 유효성을 진단합니다.)
  • - 웹 개방성 진단 (robots.txt 구문 및 구조를 검사하고 해당 URL의 접근을 허용하는지 진단합니다.)

웹 접근성 - A=CollCheck

A=CollChecksms!은 홈페이지 관리자, 개발자, 디자이너 등의 입장에서 접근성을 검사하고 관리할 수 있도록 하는 '웹 접근성 통합/관리 솔루션'이다.

A=CollCheck 다운로드 A=CollCheck 사용방법

주요 기능

  • - 웹 접근성 자동검사
  • - 웹 접근성 검사결과 보고서 제공
  • - 프로젝트 관리기능
  • - 웹 접근성 관련지침 및 가이드 정보 제공
  • - 웹 접근성 수동검사 편의기능 지원(오류 캡쳐 등)
